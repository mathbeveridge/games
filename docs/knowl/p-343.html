Our base case is \(m=0\text{,}\) where neither graph has any edges. Of course, this is a terminal position (hence a \(\cP\)-position). Futhermore, in this case, \([G_y:T_1] = [G_y:T_2]\text{,}\) so this is not very satisfying. So let's look and the next values of \(m\text{.}\) If \(m=1\text{,}\) then once again, the graphs are identical: they must each be bamboo stalks of height 1. This is obviously a \(\cP\)-position by Tweedledum-Tweedledee. At \(m=2\text{,}\) we finally have a non-trivial example: there are three possibilities for the position \([G_y:T_1] \oplus [G_y:T_2]\text{,}\) shown in FigureÂ 4.3.5. Each of these positions is a \(\cP\)-position. Only the third example has \(T_1 \neq T_2\text{.}\)
