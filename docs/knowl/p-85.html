Borrowing the multiset notation from Nimble, we use \(\{ n_1, n_2, \ldots. n_k \}\) to represent a Nim game with \(k\) heaps, where the \(i\)th heap contains \(n_i\) beans, \(1 \leq i \leq k\text{.}\) For example, FigureÂ 2.1.1 shows the game position \(\{ 4, 3, 7 \}.\) There are 14 possible moves from this position. Are any of these options winning moves? We will soon see that the answer to this question is "No." This Nim position is a \(\cP\)-position.
